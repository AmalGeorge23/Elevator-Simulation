Elevator controllers are fundamental digital control systems that ensure safe, reliable, and efficient transportation in multi-floor buildings by coordinating request handling, motion sequencing, and door operation under deterministic timing constraints. This project presents the design and simulation of a multi-floor Elevator Controller using Verilog HDL, implemented as a finite state machine (FSM) with clearly defined states such as IDLE, MOVE, and DOOR to man-age floor requests, determine travel direction, update the current floor based on a configurable floor-travel timer, and execute door-open/door-close sequencing at service stops. The controller additionally incorporates safety-oriented door behavior, including door obstruction handling and manual door open/close overrides, to improve robustness under practical operating conditions. Functional verification is carried out using Icarus Verilog for compilation and simulation, and GTKWave for waveform-based analysis, validating correct servicing of multiple requests, proper up/down motion control, accurate timing behavior, and safe door operation.
